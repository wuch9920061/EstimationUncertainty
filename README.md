Codebase for performing for quantifying estimation uncertainty in parametric seismic assessment procedures including both seismic code-prescriptive assessment and seismic risk-based assessment.

This folder contains the code associated with the study:

C. Wu and H. V. Burton (2024+), Effects of probability model misspecification on the number of ground motions required for seismic performance assessment (under review).

Code will be made publicly available. We will be updating this repository shortly.
